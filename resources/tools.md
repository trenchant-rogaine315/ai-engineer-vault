# Vetted AI Engineering Tools

This list contains 40+ production-ready tools for AI Engineering. We prioritize open-source, vendor-agnostic, and well-maintained frameworks.

---

## Orchestration & Frameworks
- **[LangChain](https://github.com/langchain-ai/langchain)** | Open Source | The ubiquitous library for building LLM applications via composable chains.
- **[LlamaIndex](https://github.com/run-llama/llama_index)** | Open Source | Focused specifically on connecting LLMs to external data sources (RAG).
- **[Dify](https://github.com/langgenius/dify)** | Open Source | A self-hosted platform for visually building and deploying LLM apps and agents.
- **[AutoGen](https://github.com/microsoft/autogen)** | Open Source | Microsoft's framework for multi-agent conversation and task orchestration.

## Vector Databases
- **[Pinecone](https://www.pinecone.io/)** | Paid/SaaS | A fully managed, cloud-native vector database for high-scale semantic search.
- **[Qdrant](https://github.com/qdrant/qdrant)** | Open Source | High-performance vector database with strong support for filtering and metadata.
- **[Chroma](https://github.com/chroma-core/chroma)** | Open Source | An easy-to-use embedding database designed for local development and rapid prototyping.
- **[LanceDB](https://github.com/lancedb/lancedb)** | Open Source | A serverless vector database that can run directly on S3 or local disk.

## Observability & Tracing
- **[LangSmith](https://www.langchain.com/langsmith)** | Paid/Freemium | Trace-level observability for LangChain apps, including cost and latency tracking.
- **[Arize Phoenix](https://github.com/Arize-ai/phoenix)** | Open Source | Notebook-first observability for evaluating and tracing LLM and RAG pipelines.
- **[Langfuse](https://github.com/langfuse/langfuse)** | Open Source | Open-source alternative for tracing and managing AI application logs.

## Evals & Testing
- **[Promptfoo](https://github.com/promptfoo/promptfoo)** | Open Source | The standard CLI for matrix-testing prompts and red-teaming model outputs.
- **[RAGAS](https://github.com/explodinggradients/ragas)** | Open Source | A framework specifically designed for evaluating the context precision and recall of RAG pipelines.
- **[DeepEval](https://github.com/confident-ai/deepeval)** | Open Source | Unit testing for LLM applications with integrated support for CI/CD pipelines.

## Inference & Serving
- **[vLLM](https://github.com/vllm-project/vllm)** | Open Source | High-throughput serving engine for LLMs with PagedAttention and continuous batching.
- **[Ollama](https://github.com/ollama/ollama)** | Open Source | The easiest way to run frontier-class models (Llama 3, Mistral) on local hardware.
- **[TGI (Text Generation Inference)](https://github.com/huggingface/text-generation-inference)** | Open Source | Hugging Face's production engine for optimized LLM serving.

## Fine-tuning
- **[Unsloth](https://github.com/unslothai/unsloth)** | Open Source | 2x-5x faster fine-tuning for models like Llama 3 with 70% less memory usage.
- **[Axolotl](https://github.com/OpenAccess-AI-Collective/axolotl)** | Open Source | A unified configuration-based framework for fine-tuning diverse models.

## Security & Guardrails
- [E2B](https://github.com/e2b-dev/e2b) | Open Source | Isolated cloud sandboxes for executing model-generated code safely within enterprise networks.
- [Guardrails AI](https://github.com/guardrails-ai/guardrails) | Open Source | Structural and behavioral guardrails for forcing deterministic outputs and safety.
